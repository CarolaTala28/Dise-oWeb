Organización de proyectos Web

Inicio Editores Comparación Organización

Estructura recomendada

                    📁 Proyecto Web
                    │
                    ├── 📄 index.html
                    ├── 📄 editores.html
                    ├── 📄 comparacion.html
                    ├── 📄 organizacion.html
                    │
                    ├── 📁 css
                    │   └── 📄 estilos.css
                    │
                    ├── 📁 img
                    │    └── 🖼 imágenes
                    │
                    └── 📁 media
                        └── 🎥 videos

Funciones de cada carpeta

  • HTML: estructura del sitio.
  • CSS: diseño visual.
  • IMG: imágenes y gráficos.
  • MEDIA: videos y audios.

Conclusión

Una buena organización facilita el mantenimiento, la ampliación y el trabajo en equipo dentro de un proyecto web.
