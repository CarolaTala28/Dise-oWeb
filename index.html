Bienvenido

El diseño web es la disciplina encargada de planificar, crear y organizar sitios web, combinando aspectos visuales, estructurales y funcionales para brindar una experiencia agradable a los uruarios.

Este sitio web tiene como objetivo explicar los conceptos básicos relacionados con el diseño web, los editores utilizados para crear páginas y la organización correcta de los proyectos.

Utiliza el menú de navegación para recorrer cada sección.
