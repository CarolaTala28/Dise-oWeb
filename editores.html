¿Qué es un editor web?

Un editor web es una herramienta que permite crear, modificar y gestionar páginas web.

¿Qué es el entorno de trabajo?

El entorno de trabajo incluye la interfaz de editor, las opciones de configuración, las ventas de diseño y los paneles para administrar archivos y recursos.

Objetivo

Facilitar la construcción de sitios web utilizando herraminetas visuales o escribiendo código directamente.
